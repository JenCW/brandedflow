# Markdown Preview & Printing Guide

## Previewing Markdown Files in Cursor

### Method 1: Keyboard Shortcut (Fastest)
1. Open the markdown file (click it in the file explorer)
2. Press **`Cmd+Shift+V`** (or **`Cmd+K` then `V`**)
3. Preview opens in a split pane showing the formatted version

### Method 2: Right-Click Menu
1. Right-click the markdown file in the file explorer
2. Select **"Open Preview"** or **"Markdown: Open Preview"**

### Method 3: Command Palette
1. Press **`Cmd+Shift+P`**
2. Type **"Markdown: Open Preview"**
3. Press Enter

---

## Printing for Client Presentation

### Option 1: Print from Preview (Recommended)
1. Open preview using `Cmd+Shift+V`
2. Press **`Cmd+P`** (Print)
3. In the print dialog:
   - Select **"Save as PDF"** (for digital sharing)
   - OR select your printer (for physical printing)
   - Adjust settings (margins, headers/footers)
4. Click **"Save"** or **"Print"**

### Option 2: Export to PDF via Browser
1. Open preview (`Cmd+Shift+V`)
2. Right-click in the preview pane
3. Select **"Open in Browser"** (if available)
4. In browser: `Cmd+P` → Save as PDF

### Option 3: Copy to Word/Google Docs
1. Open preview (`Cmd+Shift+V`)
2. Select all (`Cmd+A`)
3. Copy (`Cmd+C`)
4. Paste into Word or Google Docs
5. Format and print from there

### Option 4: Use Online Converter
1. Copy the markdown content
2. Go to: https://www.markdowntopdf.com/ or https://dillinger.io/
3. Paste and convert to PDF
4. Download and print

## Pro Tips

### For Best Print Quality:
- Use **"Save as PDF"** instead of printing directly
- PDFs preserve formatting better
- Easier to email/share with clients

### For Client Emails:
- Attach the PDF
- Or use a service like Dropbox/Google Drive
- Include a brief summary in the email

### For Presentations:
- Export to PDF
- Use PDF in presentation software
- Or print and bind for in-person meetings

---

## Troubleshooting

### Preview Not Showing?
- Make sure the file is a `.md` file
- Try `Cmd+Shift+P` → "Markdown: Open Preview"
- Restart Cursor if needed

### Print Not Working?
- Make sure preview is open first
- Try opening in browser (right-click preview → "Open in Browser")
- Use online converter as backup

### Formatting Looks Wrong?
- Markdown preview should format automatically
- If issues, try the online converter method
- Or copy to Word/Google Docs for more control

---

**Remember**: The editor tab shows the raw markdown code. The preview shows the formatted version. Always use preview for printing/presenting!
