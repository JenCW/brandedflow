## How to Use This Template

1. **Fill out this template** with client information
2. **Save to:** `clients/{client-name}/01_intake/quick-intake-{date}.md`
3. **Run MCP:** `process-client-intake` with `{ client_name: "{client-name}", source: "manual", intake_data: {...} }`
4. **Profile created:** Client profile automatically created with all variations

---

**Note:** This is for smaller projects. For larger projects, use the full client intake form in Airtable.
