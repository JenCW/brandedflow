# AQ Remodeling - Fresh Build for 10Web/Elementor

## Purpose
Clean rebuild optimized for:
1. **Netlify deployment** (static hosting)
2. **10Web AI Builder conversion** (to WordPress/Elementor)
3. **Client management** (easy editing in Elementor)

## Structure
- `index.html` - Homepage
- `css/style.css` - Main stylesheet (Elementor-friendly classes)
- `js/script.js` - JavaScript
- `images/` - All images
- `DEPLOYMENT.md` - Step-by-step deployment guide

## Workflow
1. Build HTML/CSS/JS site ✅
2. Deploy to Netlify ✅
3. Use 10Web AI Builder to convert ✅
4. Apply CSS overrides ✅
5. Client edits in Elementor ✅

## Key Design Principles
- **Elementor-friendly class names** (easy to target in CSS)
- **Semantic HTML** (better conversion)
- **Clean structure** (sections, containers, widgets)
- **Minimal dependencies** (fast loading)
