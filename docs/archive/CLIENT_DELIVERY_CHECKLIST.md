## PHASE 1: KICKOFF

### Before First Meeting
- [ ] Send welcome email with meeting link
- [ ] Review their website/social media
- [ ] Prepare questions about their goals
- [ ] Create project folder (digital + physical notes)

### During Kickoff Call
- [ ] Record their business goals
- [ ] Understand target audience
- [ ] Get brand assets (logos, colors, fonts)
- [ ] Set expectations for timeline
- [ ] Confirm deliverables list
- [ ] Schedule check-in dates

### After Kickoff
- [ ] Send meeting recap email
- [ ] Request payment (if deposit required)
- [ ] Add deadlines to calendar
- [ ] Create project timeline

---

## PHASE 2: CREATION

### Design/Strategy Work
- [ ] Create first draft/concept
- [ ] Internal review (does it meet goals?)
- [ ] Prepare presentation/mockup
- [ ] Document decisions made

### Client Review
- [ ] Send draft with clear questions
- [ ] Schedule review call (don't just email)
- [ ] Take detailed notes on feedback
- [ ] Confirm scope (watch for scope creep!)

### Revisions
- [ ] Make approved changes
- [ ] Document what changed and why
- [ ] Second review if needed
- [ ] Get final approval in writing

---

## PHASE 3: DELIVERY

### Before Handoff
- [ ] Final quality check
- [ ] Prepare all files in organized folder
- [ ] Create instructions/guide if needed
- [ ] Test everything works

### Delivery Day
- [ ] Send files with clear labels
- [ ] Include usage instructions
- [ ] Explain what they're getting
- [ ] Schedule handoff call if complex

### Handoff Call
- [ ] Walk through deliverables
- [ ] Answer questions
- [ ] Explain next steps (if applicable)
- [ ] Record any post-project requests

---

## PHASE 4: WRAP UP

### Financial
- [ ] Send final invoice
- [ ] Confirm payment received
- [ ] Update financial tracker
- [ ] File receipts/expenses

### Documentation
- [ ] Save all project files to archive
- [ ] Document lessons learned
- [ ] Note what worked well
- [ ] Record time spent vs. estimated

### Client Relationship
- [ ] Send thank you note
- [ ] Request testimonial/review
- [ ] Add to portfolio (with permission)
- [ ] Set follow-up reminder (30-60 days)

### Marketing
- [ ] Case study (if applicable)
- [ ] Before/after for portfolio
- [ ] Social media post (with permission)
- [ ] Add to wins tracker

---

## RED FLAGS TO WATCH

**Stop and reassess if:**
- [ ] Scope keeps expanding without additional payment
- [ ] Client isn't responding to requests for info
- [ ] You're doing work not in original agreement
- [ ] Payment is late or disputed
- [ ] Timeline has shifted dramatically

**Action:** Schedule call to realign expectations

---

## POST-PROJECT FOLLOW-UP

**30 Days After:**
- [ ] Check in: "How's everything working?"
- [ ] Ask about results they're seeing
- [ ] Offer additional services if relevant
- [ ] Request referrals if happy

**90 Days After:**
- [ ] Quick check-in email
- [ ] Share relevant tip or resource
- [ ] Stay on their radar
