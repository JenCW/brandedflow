## When to Use Which

### Use MASTER_RULES.md When:
- ✅ Setting up a new AI platform (Cursor, Claude, ChatGPT)
- ✅ Need full context about the project
- ✅ Encountering edge cases
- ✅ Need detailed explanations
- ✅ Creating comprehensive documentation

### Use MASTER_RULES_QUICK.md When:
- ✅ Quick start in a new chat
- ✅ Just need the basics
- ✅ Copy-pasting rules into a conversation
- ✅ Reminding yourself of the essentials
- ✅ Fast reference

## Relationship

**MASTER_RULES_QUICK.md** is a condensed version of **MASTER_RULES.md**

Think of it like:
- **MASTER_RULES.md** = Full manual
- **MASTER_RULES_QUICK.md** = Quick start guide

Both should stay in sync - if you update one, update the other!
